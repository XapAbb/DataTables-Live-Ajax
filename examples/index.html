DataTables LiveAjax Plugin

Author: Justin Hyland
Links: GIT Repository | DataTables | Author | Other Projects by Author | Other Cool Stuff

Description: Keep an AJAX sourced DataTable up to date, without reloading the entire table at once (with custom interval and pausing ability)

LiveAjax Examples

  • Basic Initialization - Check for updates from the data source every 5 seconds (default)
  • Advanced Initialization - Manage the liveAjax via API Methods, and watch the activity with liveAjax events
  • Basic Initialization w/ Custom ajax.dataSrc Function - A basic example to demonstrate liveAjax working fine with the DataTables setting ajax.dataSrc defined as a custom function
